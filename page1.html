Machine Learning Sentiment Analysis

The goal of this project was to discern the sentiment of a text data using various NLP techniques to both compare their performance and glean real-life insight into our datasetâ€™s ramifications. Sentiment analysis has recently grown in popularity, used in marketing and financial forecasting, as well as for reactionary purposes in tertiary sector enterprise. With the recent growth of feedback and review channels like social media, blog posts, etc., there has been an influx of electronically stored data. It is important to analyze and categorize this data to improve customer experience. Opinions and sentiments impact our actions and behaviors, which makes sentiment analysis all the more crucial. This project explored multiple models including Random Forest, Logistic Regression, SVM, and MultinomialNB to compare and evaluate the accuracies of the results.
